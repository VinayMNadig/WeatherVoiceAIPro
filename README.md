# 📖 Overview

WeatherVoiceAI is an AI-powered weather assistant that transforms traditional weather applications into natural conversations.

Instead of manually searching forecasts, users simply ask questions like:

> "Will it rain tomorrow?"

or

> "Should I carry an umbrella for my evening walk?"

The assistant understands natural language, retrieves live weather information, analyzes conditions using AI, and responds with intelligent recommendations.

The goal is to make weather information faster, easier, and more human.

---

# 💡 Why We Built This

Most weather applications provide raw numbers:

- Temperature
- Humidity
- Wind Speed
- Pressure

But users don't really want numbers.

They want answers.

Examples:

❌ 32°C

✅ It's very hot today. Stay hydrated and avoid outdoor activities during the afternoon.

Instead of displaying weather,
WeatherVoiceAI explains weather.

---

# 🎯 Problem Statement

Traditional weather apps have several limitations:

- Difficult to interpret weather data
- No conversational interface
- Limited personalized recommendations
- No AI-generated explanations
- Poor accessibility for non-technical users

WeatherVoiceAI solves these problems by combining AI with live weather data.

# ⚙ Installation

Clone repository

```bash
git clone https://github.com/VinayMNadig/WeatherVoiceAIPro.git
```

Install dependencies

Frontend

```bash
cd frontend
npm install
```

Backend

```bash
cd backend
npm install
```

---

# 🔑 Environment Variables

Backend

```env
OPENWEATHER_API_KEY=your_key
GROQ_API_KEY=your_key
```

Frontend

```env
VITE_API_URL=http://localhost:5000
```
